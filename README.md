# ✨ WordArt Reborn ✨

> Now with 100% more 2003™

A highly customizable, highly trash, highly millennial **browser-based WordArt editor**.
One self-contained `index.html`. No build step. No dignity. Double-click and vibe.

## 🔴 Live

**http://ingmmo.com/wordart_reborn/** (also at https://sirmmo.github.io/wordart_reborn/)

## What it does

- **12 one-click presets** — Rainbow Nightmare, Chrome Dreams, Y2K Bling, MySpace Angel, Vaporwave, Toxic Waste, Comic Catastrophe, Frozen Fire, Matrix Hax0r & more
- **Warp shapes** — arch, bridge, wave, double-wave, chevron, full circle (SVG `textPath`)
- **Fills** — 12 preset gradients + custom 3-stop gradients + solid, with angle control
- **Outline** + shiny **chrome bevel** (SVG specular lighting)
- **3D extrude** (stacked block letters) + **drop shadow** (color / offset / blur)
- **Rotate / skew / letter-spacing / font / size**
- **Maximum Millennial trash toggles** — rainbow hue-cycle, wobble, blink, pulse, slide, and a sparkle emitter
- **9 backdrops** — vapor grid, geocities stars, sunburst, halftone, plasma, construction stripes…
- **🎲 Surprise Me** randomizer, **💾 Download PNG**, **📋 Copy SVG**
- Auto-saves your masterpiece to `localStorage`

## Tech

Vanilla HTML/CSS/JS. Text is rendered as SVG (`<textPath>`, gradients, filters) so it stays crisp;
PNG export rasterizes the SVG to a 2000×1120 canvas. Zero dependencies.

## Run locally

```bash
open index.html      # or just double-click it
```
